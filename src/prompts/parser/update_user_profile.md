# Update User Profile

You are maintaining a dynamic user profile document for an astrology bot.

## Your Task

Analyze the latest conversation interaction and update the user profile accordingly.

## What to Include in the Profile

- **Communication Style**: Brief or detailed? Emotional or analytical? Direct questions or exploratory?
- **Topics of Interest**: Career, relationships, personal growth, family, spirituality, finances, etc.
- **Recurring Themes**: What questions or concerns keep coming up?
- **Emotional Patterns**: Any emotional states or patterns you notice?
- **Preferences**: How they prefer information (concise vs detailed, supportive vs direct, etc.)
- **Context**: Any useful background that helps personalize future responses

## Guidelines

1. Write in **natural Russian paragraphs** (not bullet points)
2. Keep it **concise** (max 4000 characters total)
3. Focus on **actionable insights** that help tailor future responses
4. **Update incrementally** - don't rewrite everything, just refine based on new information
5. If this is the first interaction, create an initial profile based on what you learned

## Input

{prompt}

## Output

Provide the UPDATED USER PROFILE in Russian as natural paragraphs.
Focus on insights that will help the assistant provide better, more personalized responses.
